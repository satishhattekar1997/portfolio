# Satish Hattekar Portfolio

Static portfolio website for Data Engineer profile.

## Run locally

1. Open index.html directly.
2. Or run a local server:
   - python -m http.server 8000 and visit http://localhost:8000
   - 
px serve and follow the startup URL.
3. Or in VS Code: install Live Server extension and click **Open with Live Server**.

## Features

- Sticky top navigation with smooth scrolling.
- Mobile responsive menu.
- Sections: Home, About, Skills, Experience, Projects, Education, Contact.
- Download Resume button linked to esume.pdf (place your actual PDF there).

## Customize

- index.html for content and structure.
- styles.css for design.
- script.js for interactive behavior.
