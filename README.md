# Satish Hattekar Portfolio (React)

This is a React-based portfolio website built with Create React App.

## Run locally

1. 
pm install
2. 
pm start
3. Open http://localhost:3000

## Build

- 
pm run build

## Notes

- Add esume.pdf in the project root to enable resume download.
- Content is in src/App.js.
- Styles are in src/App.css.
